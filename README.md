# Jatinbansal1998 Tap

## How do I install these formulae?

`brew install jatinbansal1998/tap/<formula>`

Or `brew tap jatinbansal1998/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jatinbansal1998/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
